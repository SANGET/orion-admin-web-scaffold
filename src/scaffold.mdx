# Scroffold 脚手架应用

> 快速开启前后端分离的管理功能前端开发。

[具体业务功能开发，请看 uke-dashboard](https://github.com/SANGET/uke-dashboard.git)

import { Playground, PropsTable } from 'docz'

import ReportDemo from './template-engine/demo'

## Usage

结合模版引擎和 Navigator 的导航机制的脚手架

参考:

- [导航器](/Navigator)
- [表格模版引擎](/Report-TMPL)
